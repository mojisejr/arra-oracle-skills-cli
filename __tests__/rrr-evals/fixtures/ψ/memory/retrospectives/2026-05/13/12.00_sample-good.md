If the sample passes by being shallow, the eval framework teaches shallowness.

## Honest Feedback

1. The Q8 oracle-sync test currently only verifies retrievability — it does not verify that the result ranks within the top-5 across all three vector models distinctly. If bge-m3 returns the doc but nomic and qwen3 don't, we still pass. Friction: the eval doesn't catch model-specific drift. Next iteration should split by model.
2. The session-log consistency checks parse a .jsonl that does not exist by default. They skip cleanly, which is correct behavior, but it means CI green does NOT prove /rrr ran without the Task tool — only that the check is wired. We need a synthetic .jsonl fixture before this gate has teeth.
3. The friction-count heuristic counts bullets, not distinct frictions. A retro with five sub-bullets under one parent point would pass with count=5 while only naming one friction. The spec said ≥3 points; I implemented ≥3 bullets. Those are different. Operational friction, not strategic error — recurs whenever bullet-style retros are written.
4. The Q9 lesson/friction overlap check uses raw token-set intersection rather than the cosine-similarity threshold the brainstorm spec proposed. Cosine was rejected because adding sentence-transformers as a test dep felt heavy. The reason this felt safe was that token-overlap captures the worst-case duplication. But it will miss paraphrased duplicates.

## Lessons Learned

- Read aspirational stubs deliberately — subprocess.run("claude") in a spec is a contract for the future, not a literal instruction. Mark stubs with TODO comments so the next implementer can find them.
- skipif beats hard-require for backend-dependent tests. Existence proves the gate is wired; runtime activation proves enforcement. Treat them as separate axes.
- Fixture retros are documentation. A shallow fixture teaches shallow output. The sample-good retro should itself pass anti-rationalization Gate 2, not just Gate 1.
- Calver-shaped paths (YYYY-MM/DD) need shutil.rmtree caution — easy to glob the wrong day if you don't normalize the date fixture.
